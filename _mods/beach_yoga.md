## About This Mod

**Beach Yoga** allows Sims to practice yoga directly on the beach using beach towels.  
As a side effect, yoga mats can also be used as beach towels.

This mod was previously part of another project and has been slightly updated and released as a standalone mod for better visibility and maintenance.

## Gameplay Details

- Yoga can be performed **directly on beach towels**.
- Yoga mats can be used **as beach towels**.
- Fully integrated into existing yoga interactions.
- No custom UI and no custom strings.

Override Mod

This mod overrides core game resources instead of adding new ones.

  • High conflict risk: Any other mod that overrides the same resources will conflict with this mod.
  • One-override rule: If two mods modify the same resource, usually only one can be used safely.
  • Patch sensitive: Override mods are more likely to break after game updates.
Recommendation: test this mod alone after patches and clear localthumbcache.package.
